Данный документ являеться техническим заданием для курсового проекта GEOTRACERT студентов 5 курса Института Информационных Технологий и Управления Санкт-Петербургского Государственного Политихнического Университета. Проект представляет собой сервис для представления маршрута трафика на карте земной поверхности с выводом данных о качестве соединения на каждом этапе маршрута.

Введение

Назначение

Данное техническое задание предназначено для описания основной функциональности разрабатываемого сервиса.

Область применения

Сервис GEOTRACERT предназначен для визуального представления пути трафиика между сервером на котором исполняется код сервиса и заданной пользователем точкой назначения.

Клиент дает возможность пользователю ввести интересующий IP-адрес. Сервер выполняет команду tracert и извлекает данные о координатах ХОП. Клиент выводит на карту маркеры ХОП и их метрику.

В отличие от стандартного вывода консольной утилиты tracert, пользователю будет дана возможность увидеть не только логическую, и поэтому весьма абстрактную метрику, но и географическое взаиморасположение узлов сети. Сервис наглядно продемонстрирует пользователю расстояния которые проходит трафик, а также поможет оценить качество инфраструктуры по всему миру.

Ограничения по памяти

Клиетская часть в современный браузер с открытой клиентской частью сервиса не должен занимать более 2 Гб памяти на персональном компьютере.

Серверная часть сервиса не должна занимать более 16 Гб памяти.
